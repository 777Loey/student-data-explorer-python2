# Student Data Explorer (Python)

A beginner-friendly data analysis script that loads a CSV file, cleans the data, and outputs useful insights.

## Features
- Load CSV data into a pandas DataFrame
- Handle missing values
- Calculate summary statistics
- Create an "average_score" feature per student
- Save cleaned output + a summary report

## Tech Stack
Python, pandas

## How to Run
1. Install dependencies:
   `pip install pandas`
2. Run:
   `python main.py`

## Outputs
- `students_cleaned.csv`
- `summary.txt`

## What I Learned
- Working with dataframes
- Cleaning and transforming data
- Communicating insights clearly through outputs and documentation
